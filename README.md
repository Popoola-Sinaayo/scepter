# The Scepter

Ministry website for **The Scepter** — built from the Client-First Church & Community Figma template and personalized with the ministry brand.

## Stack

- **Next.js** (App Router) — SSR / RSC by default
- **TypeScript**
- **Tailwind CSS v4**
- **Roboto Condensed** (matches Figma typography)

## Pages

| Route | Page |
| --- | --- |
| `/` | Home |
| `/about` | About Us |
| `/sermons` | Sermons & Events |
| `/blog` | Blog |
| `/contact` | Contact |

Contact: **info@thescepterglobal**

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
